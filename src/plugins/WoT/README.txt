COMPILATION
===========

In order to compile WoT, you need to obtain db4o 7.2 for Java (http://www.db4o.com/DownloadNow.aspx)
To compile:
  $ ant -Dfreenet-cvs-snapshot.location=/path/to/freenet-cvs-snapshot.jar -Ddb4o.location=/path/to/db4o-7.2.39.10644-java5.jar


RUNNING
=======

Visit the plugins page ("http://127.0.0.1:8888/plugins/ by default) with your Web browser.

In the Load Plugin box, enter: /path/to/wot.jar and click the Load button.

Then click the Visit button.
